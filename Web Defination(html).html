Definition

1. Website

Website is a collection of web pages or one web page. Which is identified by a common domain and registered under at least one web server.
The website consists of hyperlink, image, text and document. example:- www.Facebook.com,,|www.Google.com,,|www.Amazon.com

2. WWW

WWW stands for World Wide Web, It is a most commonly used subdomain. WWW is a system which connects resources like documents and hyperlink stored anywhere in the world.

3. HTTP/HTTPS

HTTP- Hyper Text Transfer Protocol is an application protocol for web data communication. The data we send or receive over the internet using this protocol only.
HTTPS is also the same but with a secured socket layer(ssl)/transport layer security(tls) protocol. it enhances security by adding encryption in data communication

4. Client Server

Client server is relation between client and server over internet to transfer files and services in programme

5. Domain

Domain is a common address of websites over the internet. it is unique and registered on web server
for example Amazon.com, Google.com

6. Host

Host is a computer or device called server which allows other users to use resources, application or services on any network

7. Hosting

Hosting is service which allows us to publish our website over internet so anyone uses internet can access our website also, it also known as website hosting or web hosting

8. FTP

FTP(file transfer protocol) is protocol used to send files and folders over internet between computers and servers

9. Path

Path is unique address or location of files and folders located in a computer

10. Extension

Extension allows computer to know which type of file it is and which programme to run
for example if there is .html file, device will open internet browser

11. Editor/VScode

Editor used for building and debugging web and cloud applications. also known as source code editor. For example vscode(visual studio code) editor. which is made by microsoft for windows, linux and macOS.

12. URL-URI/URN

URL(Uniform Resource Locator)-all websites have unique addresses to locate them on internet these addresses called url
URI(Uniform Resource Identifier)-identifies a resource and differentiates it from others by using name, location or both. it contains components like scheme, authority, path and query URI can be URL, URN or both
URN(Uniform Resource Name)-is simply a unique name which identifies resources on the internet. URL uses network addresses where URN uses names.

13.Address bar

It is a text field on top of the web browser which shows URL of current web page

14. Title bar

It shows an icon and text on top of the window which displays the title of the current web page or document.
